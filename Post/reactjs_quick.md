# REACT JS  Y DotNET CORE RAPIDO

La manera mas rapida de generar una aplicacion de Reactjs usando dotnet core es con el command line de dotnet
Ejecutando el siguiente comando podras ver los distintos tipos de app que se puede generar:

```
C:\user\blabla> dotnet new

--------------------------------------------------------------------------------------------------------

xUnit Test Project                                xunit            [C#], F#, VB      Test/xUnit
ASP.NET Core Empty                                web              [C#], F#          Web/Empty
ASP.NET Core Web App (Model-View-Controller)      mvc              [C#], F#          Web/MVC

ASP.NET Core with Angular                         angular          [C#]              Web/MVC/SPA
```
